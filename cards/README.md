# Cards
## 學習目標
熟悉Angular基礎與法
## 學習內容
### Static Files
* 存放在assets資料夾
### Scoped CSS Files
* component為Scoped CSS(即指影響該component style)
### Create component
* ```ng generate component <name>```
### ngFor
* ```*ngFor = "let item of array"```
### CSS Selector
* 在app-component使用```:host```可選取到app-root
### Input
* child component: ```@Input() variable```
* parent component: ```[input variable] = "parent value"```
